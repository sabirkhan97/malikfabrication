# Website Redesign Plan - Malik Fabrication

## Project Overview
- **Project**: Redesign MK-FAB website to public standard
- **Brand**: Malik Fabrication
- **Theme**: Light theme (professional, clean)
- **Structure**: Multi-page components
- **Color Scheme**: Professional Blue (#1e40af) with Orange (#e05a00) accents

## Design Decisions
- **Font**: Modern clean fonts (Inter, Poppins)
